# Beginner Project 25: Personal Finance Tracker With Forecasts

**Objective:** Ingest transactions, categorize spending, and forecast cashflow.

## Folder Structure

- main.py: Entry point and CLI/application flow
- models.py: Data models and constructors
- storage.py: File/database persistence helpers
- operations.py: Core business logic
- display.py: Output formatting and user-facing presentation

## Getting Started

1. Create a Python virtual environment.
2. Install project dependencies for this project.
3. Implement each TODO module in this build order:
   - storage.py
   - models.py
   - operations.py
   - display.py
   - main.py

## Done Criteria

- The project runs from main.py without crashes.
- Core workflows from the project instructions are implemented.
- Outputs are readable and validated with sample input.
