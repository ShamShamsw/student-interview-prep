# 05. Contains Duplicate

Difficulty: Easy  
Topics: array, hash-set

Statement
Given an integer array `nums`, return `true` if any value appears at least twice; otherwise return `false`.

Examples
- Input: `nums = [1,2,3,1]`  
  Output: `true`

Constraints
- `1 <= nums.length <= 10^5`
- `-10^9 <= nums[i] <= 10^9`

Hints (optional)
- Compare length of set and original list.
