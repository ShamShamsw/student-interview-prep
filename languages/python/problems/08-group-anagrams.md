# 08. Group Anagrams

Difficulty: Medium  
Topics: array, string, hash-map

Statement
Given an array of strings `strs`, group the anagrams together. Return the grouped lists in any order.

Examples
- Input: `strs = ["eat","tea","tan","ate","nat","bat"]`  
  Output: `[["eat","tea","ate"],["tan","nat"],["bat"]]`

Constraints
- `1 <= strs.length <= 10^4`
- `0 <= strs[i].length <= 100`

Hints (optional)
- Use sorted string or character count tuple as the hash key.
