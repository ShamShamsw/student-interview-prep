# Language Tracks

This folder groups all language-specific interview prep content.

## Choose A Track

- [Python](python/README.md): Primary track with the most complete content (problems, beginner projects, mini-projects, and larger projects)
- [JavaScript](javascript/README.md): Interview problems with Jest tests and linting workflow
- [SQL](sql/README.md): Query-focused interview practice from foundational SQL to advanced analytics patterns

## Recommended Order

1. Start with Python for broad interview problem coverage.
2. Add JavaScript if you interview in frontend/full-stack roles.
3. Add SQL if you target backend, data, analytics, or product-focused roles.

## Shared Principles Across All Tracks

- Solve for correctness first, then optimize.
- Write down complexity for every solution.
- Test edge cases explicitly.
- Keep explanations concise and implementation-focused.
- Practice consistency: file naming, folder structure, and reproducible commands.
